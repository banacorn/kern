redisev

Install

npm install redisev

Usage

var redisev = require('redisev');
var client = redisev.create();
client.connect();

SET and GET

client.set('lemon', 'some lemon');
client.get('lemon', function (err, res) {
    console.log(res);
});
some lemon

client

How about some events

client
    .get('lemon')
    .get('sugar')
    .collect('ingredients ready');

client.on('ingredients ready', function(err, res){
        console.log(res);
        makeLemonade(res);                    
});

Methods

see Helper Functions if you're looking for something likeSETGET

redisev.create()

Creates a client

var redisev = require('redisev');
var client = redisev.create();

client.connect([port=6379], [host='localhost'], [auth])

Establish a connection and authenticate to the server

client.connect(6379, '140.113.123.123');

client.disconnect()

Quit and disconnect from the server

client.disconnect();

Helper Functions a.k.a commands

Basically all redis commands are supported, which comes in like: 

client.command(arg0, arg1, arg2... , [callback])

e.g.

client.quit();
client.incr(key);
client.set(key, value);
client.ping(callback);
client.get(key, callback);
